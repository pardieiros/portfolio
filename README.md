# Portfolio

React + TypeScript portfolio using Tailwind CSS and i18n (react-i18next).  
Images are referenced from `/img/**` already present in this repository.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Build:
   ```bash
   npm run build
   npm run preview
   ```

## i18n
- Default language: `pt`
- Toggle language from the top-right button (`PT`/`EN`).

## Structure
- `src/components/*`: UI sections (Navbar, Hero, PortfolioGrid, Testimonials)
- `src/i18n.ts`: language resources (EN/PT)
- `src/styles.css`: Tailwind and custom utility classes
